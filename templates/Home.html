PROJECT RUDRA

This project aims to develop a Deep-Learning based application to estimate intensity of cyclones using INSAT-3D IR imagery.
The successful implementation of system can help immensely to observe and understand the instantaneous structural changes during evolution, intensification, and landfall of Tropical Cyclones.
Through traditional techniques, predicting accurate center of the cyclone is difficult and has low precision and accuracy. To help resolve this, the center can be identified accurately using the CNN based model with high levels of accuracy.

Cyclone Detection

This module is used for detection of the presence of cyclone in the IR image obtained from the satellite.

Cyclone Intensity Estimation

This module is used for estimating the intensity of the cyclone from the latest images retrieved.

Current Analysis

This module is used for analysing the intensities estimated from the recently processed images.
